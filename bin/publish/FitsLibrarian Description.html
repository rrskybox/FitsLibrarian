FitsLibrarian Description
Browse chooses the root directory for FITS les to be viewed/edited. All les stored in this directory and
all subdirectories will be collated viewing and/or editing. Once selected, the left hand frame will show
the directory structure and location of FITS les in those directories.
Initially all les will be parsed for header information. The available header elds will be posted in the
middle frame. All les may not contain all header elds.
Initially, all le names will appear in the right-most frame grid.
Directory Frame (leftmost frame):
Selection of individual directories will limit viewing/editing to only those FITS les. Selection of a
single le will limit viewing/editing to that le.
Field Frame (middle frame):
Selection of eld header names will result in display of those elds and their values in the right-
most frame grid, by le name.
Data Frame (right-most frame)
Field values can be edited by selecting a cell value and editing. Editing is terminated by the Enter
key. Upon termination, the edited value will be changed in the associated le, if veried. Field
values for multiple les can be modied by selecting multiple values for a eld (Shift Click or
Control Click) then editing the highlighted value.
With the exception of the FITS le name column, the grid can be sorted alphanumerically
according to header eld by clicking on the associated column header.
Clicking on any le name or le name header cell opens the Add/Delete window. If an individual
le name is selected, then the Add/Delete will apply only to that FITS le. If all les are selected,
then the Add/Delete will apply to all les listed.
Initially, Add or Delete must be chosen. If Add is chosen, then a Field Name can be entered and
(optionally) a value for that eld. The new eld with the new value will be created in all selected les
upon clicking on the OK button. If Delete is chosen, then that eld (and its associated value) will be
deleted from all selected les upon clicking on the OK button. The Add/Delete window will be closed
with no action if Cancel is selected.
